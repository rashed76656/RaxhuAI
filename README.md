# 🤖 RaxhuAI — Interactive AI-Powered Portfolio

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen?style=for-the-badge&logo=vercel)](https://raxhu.bro.bd)
[![GitHub Repo](https://img.shields.io/badge/GitHub-Repo-blue?style=for-the-badge&logo=github)](https://github.com/rashed76656/RaxhuAI.git)

Welcome to **RaxhuAI**, a futuristic, chatbot-style interactive portfolio designed for **Md Rashed Mahmud**. This isn't just a website; it's a living digital experience where an AI-powered character guides you through Rashed's skills, projects, and achievements.

> [!IMPORTANT]
> **RaxhuAI** is built with a "Vibe Coding" philosophy — merging AI intelligence with creative front-end design to craft something truly unique and interactive.

---

## ✨ Key Features

-   **🎭 Interactive AI Character**: A fully animated character named Rashed (AI version) that responds to your input with multiple expressions: `idle`, `waving`, `talking`, `excited`, `thinking`, `mysterious`, `sleeping`, and even `blushing`.
-   **💬 Master Command System**: Navigate the entire portfolio using a powerful terminal-style command system. Try typing `skills`, `projects`, or `contact`.
-   **🇧🇩 Banglish & Casual Support**: The AI understands 160+ commands, including native Banglish expressions like `ki khobor`, `load sheddhing`, `sigma`, `rizz`, and many more!
-   **🥚 Easter Eggs & Hidden Secrets**: Try secret codes like `sudo hire rashed`, `hack`, `konami`, or `power up` to unlock special animations and effects.
-   **✨ Glassmorphism UI**: A premium, modern interface with sleek dark-mode aesthetics, blurred backgrounds, and glowing gradients.
-   **📱 Fully Responsive**: Optimized for every device, from high-end desktops to mobile screens.
-   **🔒 Cybersecurity Focused**: Reflects Rashed's passion for cybersecurity with themed "hacking" effects and security-related Easter eggs.

---

## 🛠️ Tech Stack

-   **Core**: Semantic HTML5 & Modern Vanilla JavaScript (ES6+)
-   **Styling**: Pure CSS3 with Custom Properties (CSS variables) for easy theme management.
-   **Animations**: Custom CSS Keyframes & SVG-based character animations.
-   **Icons**: [Lucide Icons](https://lucide.dev/) for UI elements & [FontAwesome](https://fontawesome.com/) for brand social icons.
-   **Typography**: Google Fonts (Inter, Orbitron for that futuristic vibe).
-   **Deployment**: Vercel (or any static hosting).

## 🚀 Getting Started

To run this project locally, follow these simple steps:

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/rashed76656/RaxhuAI.git
    ```

2.  **Navigate to the Directory**:
    ```bash
    cd RaxhuAI
    ```

3.  **Run Locally**:
    Since this is a vanilla project, you can simply open `index.html` in your browser. Alternatively, use a local server for the best experience:
    -   If using VS Code, install the **Live Server** extension and click "Go Live".
    -   Or use Python: `python -m http.server 8000`.
